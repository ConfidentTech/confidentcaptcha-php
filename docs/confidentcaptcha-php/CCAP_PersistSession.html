Description | Methods (details)

Session-based persistence for Confident CAPTCHA Policies

Uses PHP's $_SESSION to store Confident CAPTCHA state. All functions require a session. Either call session_start() at the top of a PHP page that will use CAPTCHA, or, better yet, call CCAP_Policy->start_captcha_page() at the top of the PHP page, before any output or headers are sent.
